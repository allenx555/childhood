## 开发环境

### 本地测试

#### 前端运行

    git clone https://github.com/allenx555/JoTang-News-KnowledgeGraph.git
    yarn # or npm i
    yarn dev # or npm run dev


#### 后端运行


建议先进入虚拟环境，这里以 virtualenv 为例：

    cd api
    # source venv/bin/activate
    pip install -r requirements.txt
